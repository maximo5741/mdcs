How to Install Lamp on Ubuntu

1) The first step is to make sure that your computer is up today by running this commandsudo apt-get update.

2) After using that command, you have to install apache sudo apt-get install apache2

3) Then, we can install the mysql server. sudo apt-get install mysql-server

4) After installing mysql, you have to install php. sudo apt-get install libapache2-mod-php php

5) Finally, after installing all of that you have to restart your apache by using this code. sudo /etc/init.d/apache2 restart

After installing lamp, the www folder should be in that location /var/www/html. In order to get there, you should us the command cd. also, if you want to add or modify content you should change the permissions by using the command.sudo chmod 755 www/
